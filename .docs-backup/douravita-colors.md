# Paleta de Cores DouraVita

## Cores Principais

### Azul (Confiança, serenidade, profundidade)
- **Primário**: #418ecb (RGB: 65, 142, 203)
- **Secundário**: #004f80 (RGB: 0, 79, 128)
- **Uso**: Textos principais, elementos de interface
- **Emoção**: Tranquilidade e credibilidade

### Verde (Sabedoria, experiência, dignidade)
- **Primário**: #00a67d (RGB: 0, 166, 125)
- **Secundário**: #005b3e (RGB: 0, 91, 62)
- **Uso**: Destaques especiais, elementos premium
- **Emoção**: Sofisticação respeitosa

### Roxo (Otimismo, energia, vitalidade)
- **Primário**: #5a4b99 (RGB: 90, 75, 153)
- **Secundário**: #40277e (RGB: 64, 39, 126)
- **Uso**: CTAs, elementos de destaque positivo
- **Emoção**: Esperança e renovação

### Amarelo (Crescimento, saúde, natureza)
- **Primário**: #efd565 (RGB: 239, 213, 101)
- **Secundário**: #f1b82c (RGB: 241, 184, 44)
- **Uso**: Indicadores de progresso, elementos de saúde
- **Emoção**: Renovação e bem-estar

## Filosofia da Paleta
"Cada cor representa uma faceta da jornada natural - do nascer ao pôr do sol, da base da montanha ao seu topo, sempre conectada com a natureza que nos fortalece."

## Aplicação no Dashboard
- **Background**: Azul escuro (#004f80) como base
- **Cards/Glassmorphism**: Verde (#00a67d) com transparência
- **Acentos positivos**: Amarelo (#efd565) para indicadores de sucesso
- **CTAs e destaques**: Roxo (#5a4b99)
- **Textos**: Azul claro (#418ecb) para títulos
